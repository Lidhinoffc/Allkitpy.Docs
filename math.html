➗ Math Module

AllkitPy Math module provides arithmetic, advanced math, geometry, statistics, and randomness utilities.

🧮 Basic Functions

Core arithmetic operations.

add()

Add two numbers.

Code

import allkitpy
print(allkitpy.math.add(10, 5))

Output

15

subtract()

Subtract second number from first.

print(allkitpy.math.subtract(10, 5))

5

multiply()

print(allkitpy.math.multiply(4, 3))

12

divide()

print(allkitpy.math.divide(10, 2))

5.0

⚡ Advanced Functions

Advanced mathematical utilities for deeper calculations.

square_root()

Returns square root of a number.

Code

import allkitpy
print(allkitpy.math.square_root(25))

Output

5.0

cube_root()

Returns cube root of a number.

print(allkitpy.math.cube_root(27))

3.0

factorial()

Calculates factorial of a number.

print(allkitpy.math.factorial(5))

120

gcd()

Finds greatest common divisor of two numbers.

print(allkitpy.math.gcd(12, 18))

6

lcm()

Finds least common multiple of two numbers.

print(allkitpy.math.lcm(4, 6))

12

absolute()

Returns absolute value of a number.

print(allkitpy.math.absolute(-10))

10

round_number()

Rounds number to given decimal places.

print(allkitpy.math.round_number(3.14159, 2))

3.14

78.53981633974483

31.41592653589793

rectangle_area()

rectangle_perimeter()

triangle_area()

Calculates area of a triangle.

print(allkitpy.math.triangle_area(10, 5))

📊 Statistics Functions

Functions for analyzing numerical datasets.

average()

Calculates mean value of a list.

Code

import allkitpy
print(allkitpy.math.average([10, 20, 30]))

Output

20

median()

Finds middle value in dataset.

print(allkitpy.math.median([10, 20, 30]))

20

mode()

Returns most frequent value in dataset.

print(allkitpy.math.mode([10, 10, 20, 30]))

10

minimum()

Returns smallest value.

print(allkitpy.math.minimum([10, 20, 5]))

5

maximum()

Returns largest value.

print(allkitpy.math.maximum([10, 20, 5]))

20

total()

Returns sum of all values.

print(allkitpy.math.total([10, 20, 30]))

60

🎲 Random Functions

Utilities for generating random values and selections.

random_int()

Generates a random integer between two values.

Code

import allkitpy
print(allkitpy.math.random_int(1, 10))

Output

7

random_float()

Generates a random float between 0 and 1.

print(allkitpy.math.random_float())

0.48291

choice()

Picks a random element from a list.

print(allkitpy.math.choice([1, 2, 3, 4]))

3

shuffle()

Randomly shuffles a list.

print(allkitpy.math.shuffle([1, 2, 3, 4]))

[3, 1, 4, 2]
